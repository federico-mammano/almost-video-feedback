# Almost Video Feedback — install (about 1 minute)

1. **Unzip** this file to a permanent spot (e.g. Documents). Don't delete the folder afterward — Chrome loads the extension from it.
2. Go to **chrome://extensions**
3. Turn on **Developer mode** (top-right toggle).
4. Click **Load unpacked** and pick the unzipped **`almost-video-feedback`** folder.
5. Pin it: click the puzzle-piece icon in the toolbar, then the pin next to "Almost Video Feedback".

## Using it

1. Open the web page you want to review (any normal http/https tab — including localhost).
2. Click the **Almost Video Feedback** toolbar icon → **Start recording**. **Allow the microphone** when Chrome asks (once per site).
3. Talk through your feedback while you click, scroll, select text, or circle the mouse around things — screenshots are captured automatically.
4. Click the toolbar icon again (or **Stop** on the on-page bar) to finish. The menu shows it saved and **copies an agent prompt + file path to your clipboard**.
5. Paste that into **Claude Code** (or your AI agent). Or open **🎥 Recordings → Share** to email / send the bundle as a `.zip`.

## Notes

- Doesn't work on `chrome://` pages or the Chrome Web Store — use a regular site.
- If **Developer mode** / **Load unpacked** is blocked by your company's IT policy, this method won't work; ask me and I'll publish it to the Chrome Web Store instead.
